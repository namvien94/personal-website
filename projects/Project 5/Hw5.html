from sklearn.model_selection import train_test_split

X = df[['X_1', 'X_2']]  # Select features (X_1 and X_2)

y = df['Y']  # Select the target variable (Y)

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)

In[6]:
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# Create an SVM classifier with a linear kernel
svm_classifier = SVC(kernel='linear')

# Train the classifier on the training data
svm_classifier.fit(X_train, y_train)

# Make predictions on the testing data
y_pred = svm_classifier.predict(X_test)

# Evaluate the accuracy
accuracy = accuracy_score(y_test, y_pred)
print(f"The Testing Accuracy is {accuracy}")

The Testing Accuracy is 0.6

Explanation:

The linear SVM's accuracy (0.6) is considerably lower than the Gaussian kernel SVM's accuracy (0.97), which strongly suggests that the data is not linearly separable, confirming by the visualization as well. That's why the Gaussian kernel's ability to model non-linear relationships allows it to achieve better performance in such cases.
